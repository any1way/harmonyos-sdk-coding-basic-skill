ArkTS基础类库是一个功能齐全的API集合，设计了一系列关键且实用的功能模块。

ArkTS基础类库主要提供了[XML生成解析转换](../XML生成、解析与转换/XML概述/xml-overview.md)、[二进制Buffer](../Buffer与FastBuffer/buffer.md)、[多种容器类库](../ArkTS容器类库/容器类库概述/container-overview.md)、[URL字符串解析](<../../../../../harmonyos-references/ArkTS（方舟编程语言）/ArkTS API/@ohos.url (URL字符串解析)/js-apis-url.md>)和[高精度浮点计算](<../../../../../harmonyos-references/ArkTS（方舟编程语言）/ArkTS API/@arkts.math.Decimal (高精度数学库Decimal)/js-apis-arkts-decimal.md>)等能力，简化开发工作，提升开发效率。
