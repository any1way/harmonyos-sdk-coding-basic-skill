CodeGenie支持在**对话框中**输入对代码段和代码文件分析要求，帮助开发者快速理解代码逻辑、代码功能、技术细节和潜在问题等，提升开发效率。

DevEco Studio 6.0.2 Beta1之前版本，分析代码文件时需要通过**Files**入口选中文件；分析代码片段时，选中代码段后需点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/78/v3/F13EC_yiSseERUZ25diloQ/zh-cn_image_0000002602185983.png?HW-CC-KV=V1&HW-CC-Date=20260611T072220Z&HW-CC-Expire=86400&HW-CC-Sign=DDD051B1D63577668F40D7335EF9D71D2806E31E0D71B900F67A67CBF895BA7D)图标开启光标上下文功能。

在DevEco Studio 6.0.2 Beta1版本，分析代码文件时，支持在对话框输入要分析的代码文件或直接分析当前文件；分析代码片段时，选中代码段后直接分析，无需开启![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6d/v3/yS6Uv2uASQesLpj47GLYLA/zh-cn_image_0000002602065927.png?HW-CC-KV=V1&HW-CC-Date=20260611T072220Z&HW-CC-Expire=86400&HW-CC-Sign=467197D225EBC1B1F44B12BAF425D57351BD43BA45A42BEF52C47C0C7088F6BE)图标。

从DevEco Studio 6.0.2 Release版本开始，使用HarmonyOS Ask智能体分析代码文件。

以DevEco Studio 6.1.0 Release和DevEco Studio 6.0.1 Beta1版本为例说明，操作如下。

**DevEco Studio 6.1.0 Release版本**

1. 选择HarmonyOS Ask智能体，在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，选择需要分析的代码文件，或在对话框输入文件路径指定需要分析的代码文件，或分析当前代码文件。
2. 在对话框输入描述，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b9/v3/EAgbBoogR9y2CxwUg88dtA/zh-cn_image_0000002602065925.png?HW-CC-KV=V1&HW-CC-Date=20260611T072220Z&HW-CC-Expire=86400&HW-CC-Sign=58F60BCC29A7356C11B745CB4E5F0B8FBD8EF9FB42A52447DEC994AE8F0EA578)发送后等待回复。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9b/v3/YVjUctOkRfCTUwr4jLkYJw/zh-cn_image_0000002571546448.gif?HW-CC-KV=V1&HW-CC-Date=20260611T072220Z&HW-CC-Expire=86400&HW-CC-Sign=035BC54DEB43FD933E11B1C6398211C9112D7F8F4F3D9B311FC70A8FB2C9B45F)

**DevEco Studio 6.0.1 Beta1版本**

分析代码文件：在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，可对单个或多个代码文件进行分析。

分析代码片段：选中代码段，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/cf/v3/JzsD5qvkSqGC1LXYdAHpEg/zh-cn_image_0000002571546452.png?HW-CC-KV=V1&HW-CC-Date=20260611T072220Z&HW-CC-Expire=86400&HW-CC-Sign=3D2C83AB8408706927D1F76EAD4477651016008247351C8777FB0319BC06D5DC)图标开启光标上下文功能，在对话框内描述，开始分析。
