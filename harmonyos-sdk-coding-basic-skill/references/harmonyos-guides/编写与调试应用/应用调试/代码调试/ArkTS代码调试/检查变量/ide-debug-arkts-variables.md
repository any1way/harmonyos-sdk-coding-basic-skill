当应用停止在某个断点处时，您可以在"Debugger"窗口中查看当前的变量信息。在"Frame"窗格中选择某个帧之后，可以在"Variable"窗格中检查变量，或对变量进行计算。

如需向"Watches"列表中添加变量或表达式，请按以下步骤操作：在"Watches"窗口中输入表达式，然后点击Add to Watches图标![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/61/v3/cXAd3huIQsivd9T95-S4Ig/zh-cn_image_0000002602066019.png?HW-CC-KV=V1&HW-CC-Date=20260611T072859Z&HW-CC-Expire=86400&HW-CC-Sign=3F8A203E0D65083DBD2044E7737EF53A9190327AFDDFE12BE495914C037A5E1B)。

如需从"Watches"列表中移除某一项，点击鼠标右键，选择**Remove Watches**。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fb/v3/S--iP9c8RReo6-tMwDsb0A/zh-cn_image_0000002602186071.png?HW-CC-KV=V1&HW-CC-Date=20260611T072859Z&HW-CC-Expire=86400&HW-CC-Sign=EEAC31AC299AA23F1F8D01F9DCF708B1105BA66211F50F2B78A90D9F720045C8)

从DevEco Studio 6.0.2 Beta1版本开始，支持预览pixelMap类型的变量。点击pixelMap变量右侧的**View PixelMap**，即可预览pixelMap。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e6/v3/XjDz89zfR_y-I4iUB0Pj6g/zh-cn_image_0000002571546540.png?HW-CC-KV=V1&HW-CC-Date=20260611T072859Z&HW-CC-Expire=86400&HW-CC-Sign=A42119CA2A6F1D245384080B8560BBDF80D2079F81243D11ADE5DEDA3AC84C58)
