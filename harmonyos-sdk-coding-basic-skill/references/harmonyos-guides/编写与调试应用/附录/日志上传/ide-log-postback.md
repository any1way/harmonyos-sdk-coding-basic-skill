说明

该功能仅支持中国境内（香港特别行政区、澳门特别行政区、中国台湾除外）。

若开发过程中遇到DevEco Studio卡顿、卡死或其他故障时，可点击IDE Error问题弹窗中Send Report，点击OK后向DevEco Studio回传日志信息。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/62/v3/SEfALTbkTfWjlq2_SaWzeA/zh-cn_image_0000002602066649.png?HW-CC-KV=V1&HW-CC-Date=20260611T073011Z&HW-CC-Expire=86400&HW-CC-Sign=866C22F34EB190D088B83D8C64AD2AC5BAF78A7402BBA3D8F353E06DE3F99473)

或通过菜单栏**Help > Collect Logs and Diagnostic Data**，选择并上传相关日志，帮助DevEco Studio提升稳定性体验。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/af/v3/Gfc20qHMRGCye6BKTuxoig/zh-cn_image_0000002571387532.png?HW-CC-KV=V1&HW-CC-Date=20260611T073011Z&HW-CC-Expire=86400&HW-CC-Sign=4EA1CB2032292873AF5277DEC40F3C6F94CBECC66F3356896CD9943CFC691B09 "点击放大")

若开发者后续需要关闭数据采集功能，请在**File > Settings**（macOS为**DevEco Studio > Preferences/Settings**）**> Appearance & Behavior > System Settings > Data Sharing**设置界面，取消勾选**Send usage statistics**关闭数据采集开关。
