Game Controller Kit（游戏控制器服务）支持游戏适配控制器外设（当前仅支持手柄），解决玩家操控性问题，保障用户体验。游戏开发者可通过接入该服务轻松实现游戏外设的上下线和按键及轴事件监听等功能。

## 能力范围

Game Controller Kit支持的能力包括:

* 监听设备上下线，查询所有在线设备的具体信息
* 监听游戏手柄的轴和按键事件

## 模拟器支持情况

本Kit暂不支持模拟器。
