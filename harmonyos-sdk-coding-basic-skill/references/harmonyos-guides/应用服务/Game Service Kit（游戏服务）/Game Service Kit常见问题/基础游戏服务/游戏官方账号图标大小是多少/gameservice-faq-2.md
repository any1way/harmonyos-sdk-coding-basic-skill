游戏官方账号图标大小总和不超过35KB。若超过35KB，调用[unionLogin](<../../../../../../harmonyos-references/Game Service Kit（游戏服务）/ArkTS API/gamePlayer（基础游戏服务）/gameservice-gameplayer.md#gameplayerunionlogin>)接口时，会出现[1002000001](<../../../../../../harmonyos-references/Game Service Kit（游戏服务）/ArkTS API/ArkTS API错误码/gameservice-error-code.md#section1002000001-游戏内部通用错误>)错误码。
