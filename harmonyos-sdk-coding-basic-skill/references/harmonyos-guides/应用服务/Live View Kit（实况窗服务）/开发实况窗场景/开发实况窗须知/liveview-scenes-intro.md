当前实况窗场景请参见[实况窗支持对接的场景](<../../Live View Kit简介/liveview-introduction.md#实况窗支持对接的场景>)。针对所有开放的场景，我们为开发者提供了实况窗[场景节点设计参考](../../实况窗设计规范/liveview-design-formula.md#场景节点设计要求)，开发者可以根据参考设计场景内节点的数量与内容。

注意

应用发送的实况窗需遵循《[实况窗设计规范](../../实况窗设计规范/liveview-design-formula.md)》，不符合设计规范的方案将不被予以开通正式权限。同时若应用实况窗上线后出现违反实况窗设计规范的行为，将被视为[违规](<../../Live View Kit简介/liveview-introduction.md#违规处罚>)。
