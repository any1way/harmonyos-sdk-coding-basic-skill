在DevEco Studio上，参考[创建一个新的工程](../../../../开发环境搭建/工程创建/创建一个新的工程/ide-create-new-project.md)和[配置调试签名（自动签名）](../../../../编写与调试应用/配置调试签名/ide-signing.md#section18815157237)完成工程应用的创建及签名配置，从而确保能顺利完成项目的编译与开发。

说明

如果开发者DevEco Studio版本低于6.0.0.848，请使用[手动签名](../../../../编写与调试应用/配置调试签名/ide-signing.md#section297715173233)完成工程应用的签名配置。
