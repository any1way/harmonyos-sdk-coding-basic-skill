当前支持的续费周期为1周、1个月、2个月、3个月、6个月、1年、30天（上次订阅日期+30天）、31天（上次订阅日期+31天），具体配置方法请参考[配置自动续期订阅商品](../../数字商品服务/配置数字商品/数字商品介绍/store-iap-product.md#配置自动续期订阅商品)。
