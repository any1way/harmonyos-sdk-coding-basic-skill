开发预加载，总体流程如下：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f/v3/Fr2rIEvDR0WMP1MBNk5ZJw/zh-cn_image_0000002622698863.png?HW-CC-KV=V1&HW-CC-Date=20260611T070600Z&HW-CC-Expire=86400&HW-CC-Sign=0F282EAAC4418FE7BAE491DCE8289F373C8F7D124202BC1171CBEB319FEB1B07)

1. [配置预加载](../配置预加载/cloudfoundation-prefetch-config.md)：在AGC云端预加载界面选择数据来源类型，并进行相关配置。
2. [开发预加载资源接口](../开发预加载资源接口/cloudfoundation-prefetch-cloud-interdev.md)：根据选择的数据来源类型，开发对应的云侧接口，以提供预加载所需的资源数据。
3. [调用预加载](../调用预加载/调用安装预加载/cloudfoundation-call-installprefetch.md)：在工程代码文件中调用预加载。
