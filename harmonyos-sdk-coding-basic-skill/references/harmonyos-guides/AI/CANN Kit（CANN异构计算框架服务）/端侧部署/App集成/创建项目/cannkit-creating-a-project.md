本章以[Caffe SqueezeNet](https://github.com/forresti/SqueezeNet)模型集成为例，说明App集成操作过程。

1. 创建DevEco Studio项目，选择“Native C++”模板，点击“Next”。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/05/v3/PLKqjA9hTFeH5Mn0_yB8Ig/zh-cn_image_0000002592379590.png?HW-CC-KV=V1&HW-CC-Date=20260611T071639Z&HW-CC-Expire=86400&HW-CC-Sign=9016DE9993FE3FA04E62766A2A59AFC2D82D8BF3B538177B07FD599607D22B51)
2. 按需填写“Project name”、“Save location”和“Module name”，选择“Compile SDK”为“5.1.0(18)”及以上版本，点击“Finish”。
