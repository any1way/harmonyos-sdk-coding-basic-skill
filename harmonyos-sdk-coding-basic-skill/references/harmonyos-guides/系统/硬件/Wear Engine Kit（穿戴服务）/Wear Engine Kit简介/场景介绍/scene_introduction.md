## 手机与穿戴设备分布式协同

* 重要通知实现腕上提醒，实现即时通知推送，如：在手机侧App中设置日程提醒、用药提醒、任务提醒等数据，可以同步到穿戴设备侧App中，不用打开手机，也可以随时在穿戴设备上查看重要信息。

  ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/eb/v3/pRlTnI3xROKYJBPfiS9Z4Q/zh-cn_image_0000002622858353.png?HW-CC-KV=V1&HW-CC-Date=20260611T065154Z&HW-CC-Expire=86400&HW-CC-Sign=9422EC5CBA029C3AD73CCEF04140A5078E44810FA921BD32D49976B4A0E02799)
* 为手机应用带来全新的“腕上交互”，如：用手机侧App看视频、听音频时，可以通过穿戴设备侧App操控手机侧App，比如暂停，下一首，停止等。

  ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a8/v3/XVzwAhdmTKyf2h43gdFOcA/zh-cn_image_0000002622698475.png?HW-CC-KV=V1&HW-CC-Date=20260611T065154Z&HW-CC-Expire=86400&HW-CC-Sign=C7C133BBBE23C9A44CEB2F5A57CAE4651BCC6CFFA194BBF0E48B4B7E4DA4076A)
* 手机侧和穿戴侧的实时协同体验。如：在手机侧App开启导航，可以通过穿戴设备侧App提示用户左转、直行、右转等。在走路或骑行时，不用每次拿出手机查看路线。

  ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e2/v3/uILQhiTUQfGk-XB5iNIebQ/zh-cn_image_0000002592218914.png?HW-CC-KV=V1&HW-CC-Date=20260611T065154Z&HW-CC-Expire=86400&HW-CC-Sign=3FB483CDFB3E08595D23D573FE26532AC43D0BF5F0AC8BBBBE3D722E7E2A61B9)

## 手机与穿戴设备虚拟化

开发者在手机侧开发App，而不需要在穿戴设备侧开发App。

* 穿戴设备状态实时掌握，如：手机侧App实时获取穿戴设备侧连接状态、佩戴状态、电量状态等，给用户提供更多增值服务。

  ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fd/v3/xRHGPTfoTNuNI-oTTheMxw/zh-cn_image_0000002592378848.png?HW-CC-KV=V1&HW-CC-Date=20260611T065154Z&HW-CC-Expire=86400&HW-CC-Sign=8E134535050EAEF83B7F74BA933D7D73F1414C6F6A5278408BA4A1D889D3CCA1)
* 穿戴设备传感器：手机侧App读取穿戴设备的传感器数据（如ECG、PPG、ACC数据等）。

  说明

  传感器受限发布，仅限专业研究机构使用。
