## 概述

定义一个结构体，用于存储AR Engine中捕获的单帧AR图像数据，包含特定时间戳下的相机图像、追踪状态、锚点及AR相关信息。

**起始版本：** 6.1.0(23)

**相关模块：** [SpatialRecon](../../模块/SpatialRecon/capi-spatialrecon.md)
