## 111001 内部错误

PhonePC/2in1TabletTVWearable

**错误信息**

resource loading failed.

**错误描述**

该错误码在资源加载失败时被触发。

**可能原因**

路径不存在，资源不存在或者文件已损坏。

**处理步骤**

检查资源是否存在或文件是否损坏。
