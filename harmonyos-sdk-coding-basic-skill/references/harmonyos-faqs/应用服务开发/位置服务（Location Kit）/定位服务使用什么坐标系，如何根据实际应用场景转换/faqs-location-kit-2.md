定位服务使用WGS84坐标系，在国内如果要配合地图使用，需要转换为GCJ02坐标系，具体转换方法可参考Map Kit（地图服务）的[坐标纠偏](<../../../../harmonyos-guides/应用服务/Map Kit（地图服务）/地图计算工具/坐标纠偏/map-convert-coordinate.md>)。
