请检查环境是否正确，PyCharm所引用的python版本是否引用hypium及hypium\_perf的全部版本，可尝试参考[场景化性能测试](<../../../../../harmonyos-guides/应用测试/专项测试/DevEco Testing/专项测试/specialized-testing.md#section8642101711299>)章节手动安装并检查依赖包完整性。

更新场景化性能测试服务后，如果本地调试失败，请检查当前依赖包版本是否与安装包中的版本一致。
