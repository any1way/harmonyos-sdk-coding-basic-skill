**DevEco Testing资料查询：**

方式1：访问DevEco Testing客户端的任务创建页面，在右下角查看各测试服务的使用指南。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8d/v3/AXFC9hjWT3CkBDKvPYZJWg/zh-cn_image_0000002455686304.png?HW-CC-KV=V1&HW-CC-Date=20260612T024624Z&HW-CC-Expire=86400&HW-CC-Sign=BB6ADBF6F8E382419E34E8C72334B766D018519F77F53486F2FC9A3076302BB0 "点击放大")

方式2：访问华为开发者联盟官网，查看指南-应用测试-专项测试中的[DevEco Testing](<../../../../harmonyos-guides/应用测试/专项测试/DevEco Testing/deveco-testing.md>)。

**DevEco Testing Hypium资料查询：**

访问华为开发者联盟官网，查看指南-应用测试-单元测试和UI测试中的[应用UI测试（基于Python）](../../../../harmonyos-guides/应用测试/单元测试和UI测试/应用UI测试（基于Python）/hypium-python-guidelines.md)。
