**问题场景**

网页加载遇到错误时触发[onErrorReceive](<../../../../../harmonyos-references/ArkWeb（方舟Web）/ArkTS 组件/Web/事件/arkts-basic-components-web-events.md#onerrorreceive>)回调。回调函数会返回相关错误码，需要了解错误码的具体含义。

**解决措施**

相关错误码可以参考[@ohos.web.netErrorList (ArkWeb网络协议栈错误列表)](<../../../../../harmonyos-references/ArkWeb（方舟Web）/ArkTS API/@ohos.web.netErrorList (ArkWeb网络协议栈错误列表)/arkts-apis-neterrorlist.md>)。
