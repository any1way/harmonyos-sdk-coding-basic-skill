Web组件可以设置[horizontalScrollBarAccess](<../../../../../harmonyos-references/ArkWeb（方舟Web）/ArkTS 组件/Web/属性/arkts-basic-components-web-attributes.md#horizontalscrollbaraccess9>)(false)属性和[verticalScrollBarAccess](<../../../../../harmonyos-references/ArkWeb（方舟Web）/ArkTS 组件/Web/属性/arkts-basic-components-web-attributes.md#verticalscrollbaraccess9>)(false)属性分别设置横向和纵向滚动条的可见性。
