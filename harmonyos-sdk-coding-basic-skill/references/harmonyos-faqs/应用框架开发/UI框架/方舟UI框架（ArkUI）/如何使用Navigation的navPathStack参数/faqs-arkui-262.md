在已通过navPathStack获取到NavDestination集合后，可通过以下方法查询参数：

* 使用[getParamByIndex](../../../../../harmonyos-references/ArkUI（方舟UI框架）/ArkTS组件/导航与切换/Navigation/ts-basic-components-navigation.md#getparambyindex10)获取指定索引的NavDestination页面的参数信息；
* 使用[getParamByName](../../../../../harmonyos-references/ArkUI（方舟UI框架）/ArkTS组件/导航与切换/Navigation/ts-basic-components-navigation.md#getparambyname10)获取所有名为name的NavDestination页面的参数信息。
