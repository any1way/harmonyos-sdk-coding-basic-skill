**问题现象**

打开上传软件包界面后，提示”No app with the specified package name is found under the team.”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/cf/v3/iX3GXO_kQB6kT7xFKUWkMA/zh-cn_image_0000002541844403.png?HW-CC-KV=V1&HW-CC-Date=20260612T024548Z&HW-CC-Expire=86400&HW-CC-Sign=6DF2AB22B9EB15933A615DA305D78F85D8218C668FEF15A102D3584A90061BB5)

**解决措施**

因为当前账号在AppGallery Connect没有对应包名的应用，请在AppGallery Connect登录当前账号后[创建应用](https://developer.huawei.com/consumer/cn/doc/app/agc-help-createapp-0000001146718717)，再重新打开上传软件包界面。
