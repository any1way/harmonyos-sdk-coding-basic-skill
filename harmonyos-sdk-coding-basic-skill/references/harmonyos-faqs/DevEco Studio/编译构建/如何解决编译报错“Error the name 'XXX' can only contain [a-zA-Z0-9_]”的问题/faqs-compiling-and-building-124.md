**问题现象**

编译报错：文件名只能包含大小写字母、数字、下划线。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/db/v3/oP1_PJzgQriVFGHvS-B_0A/zh-cn_image_0000002194158636.png?HW-CC-KV=V1&HW-CC-Date=20260612T024244Z&HW-CC-Expire=86400&HW-CC-Sign=43F9EB576D0F2BB4A5D6BC9958D766CC1968F1AA7F8C21A66E3A820A2AA21EFD)

**解决方案**

检查文件名是否包含除了大小写字母、数字和下划线之外的其他字符。
