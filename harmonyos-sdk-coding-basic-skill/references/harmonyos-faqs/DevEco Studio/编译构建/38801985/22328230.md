**问题现象**

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b0/v3/bkwWqg7hT9WZ8Q0GCQl2MA/zh-cn_image_0000002229604385.png?HW-CC-KV=V1&HW-CC-Date=20260612T024133Z&HW-CC-Expire=86400&HW-CC-Sign=52B3A9B1AF8EE6053D88163EFB553957842D98EDACB65699520A70B5D0B8AB18 "点击放大")

**问题分析**

报该错误，可能是build-profile.json5文件中未添加“targets”配置，Module Target下为空，工程同步失败。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bc/v3/uLMlZ3K0Quutb0zqBsmkPA/zh-cn_image_0000002272490329.png?HW-CC-KV=V1&HW-CC-Date=20260612T024133Z&HW-CC-Expire=86400&HW-CC-Sign=9DCD435F34A3F9B5B43FABFAF578D6C3893DA7217E26DCE4D281F8A856C86AA6)

**解决措施**

需要在模块级build-profile.json5文件中添加“targets”配置，点击“Sync Now”，待完成同步后，即可解决该问题（确保工程同步成功）。具体配置如图所示：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/15/v3/3Zr4XoS_TTqfCeICj42Zag/zh-cn_image_0000002229758873.png?HW-CC-KV=V1&HW-CC-Date=20260612T024133Z&HW-CC-Expire=86400&HW-CC-Sign=CFC49B0AACFFBCA8EDCD6DBA46A3C2663D8535F118552398499C85A314F0168A "点击放大")
