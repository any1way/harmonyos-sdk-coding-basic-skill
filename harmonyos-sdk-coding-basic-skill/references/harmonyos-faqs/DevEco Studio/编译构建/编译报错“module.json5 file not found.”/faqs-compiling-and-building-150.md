**错误描述**

未找到module.json5文件。

**可能原因**

模块下的module.json5文件缺失。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/82/v3/7AkWBGIdSiCUZ9ILfeHRhA/zh-cn_image_0000002229604093.png?HW-CC-KV=V1&HW-CC-Date=20260612T024310Z&HW-CC-Expire=86400&HW-CC-Sign=07A8715B8A366D5C598CF315C388A0F86EBC60AE4A89109D8817DD2518EA2613)

**解决措施**

在Stage模型项目中，确保每个模块包含module.json5文件。
