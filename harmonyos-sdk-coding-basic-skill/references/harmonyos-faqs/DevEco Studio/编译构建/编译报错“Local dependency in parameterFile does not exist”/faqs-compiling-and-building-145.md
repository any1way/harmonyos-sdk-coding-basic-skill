**错误描述**

parameterFile 中的本地依赖项不存在。

**可能原因**

parameterFile中定义的值不存在。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4f/v3/yJNjYDuaR5m75UiF_G-ZeQ/zh-cn_image_0000002194318024.png?HW-CC-KV=V1&HW-CC-Date=20260612T024305Z&HW-CC-Expire=86400&HW-CC-Sign=7CA58C6A783F52B358374221D7A805E8CB158602CB916B6A265E73159EFBA1F5)

**解决措施**

检查本地依赖路径是否存在，以及是否区分大小写。
