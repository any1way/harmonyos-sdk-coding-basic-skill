**问题现象**

在不同的文件中声明相同变量、interface、enum等类型，DevEco Studio不报错，但编译时会报错。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8b/v3/f-nLS3cMTcmaT6GmeS7jgw/zh-cn_image_0000002229604153.png?HW-CC-KV=V1&HW-CC-Date=20260612T024246Z&HW-CC-Expire=86400&HW-CC-Sign=658A99FE30B37CB53276A93AA65917A0A77B07A4A14CA0EC0DDBB9E386DC7610)

**解决方案**

如果文件中不包含export关键字，该文件将视为全局命名空间的一部分。
