使用代码模板可以提高开发效率。在DevEco Studio中，可以通过“File \> Settings \> Editor \> Live Templates”查看现有模板并新增自定义模板。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/64/v3/k9HAhN1qSyCytGHFyXq2mw/zh-cn_image_0000002194318448.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=6057A4C07D08238209FAA1A00F566CEB01275DDC3C5762DC9317EFB35BE46AA5 "点击放大")

点击展开按钮查看现有模板详情，点击右上角加号新增自定义模板。

   输入button后，显示代码模板，双击自动补全。
2. 自定义代码模板。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/24/v3/2u3x3kkJQDmyD7QJSJkqJg/zh-cn_image_0000002229604221.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=F631E9AEFEF23376104BFDD91C2082689DF88EBA0ED92A7BEE4D4EFC62FFD253 "点击放大")![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d7/v3/ohXjp7o4RL6BE3U5EeX8eQ/zh-cn_image_0000002194158848.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=D2CB436DB142D6DE93467A2D8BECE4281D3E593ADB738EF0DE364C8B80DD3BC4 "点击放大")

   选择“Template Group”，新增模板群组以表示自定义模板。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/19/v3/yosFdGj2QZ6pDWoRdTz2mQ/zh-cn_image_0000002229604229.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=7D5EC1DB83D038805BD65ACD2151E8ADD008C8AAC9FDF3510DCDF8B62AC15BF0 "点击放大")

   选中自定义群组，点击+，新增模板。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/42/v3/X0KzL5ylQEGx9f9VA2I1zQ/zh-cn_image_0000002194318456.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=D1C332A1BFCF76E065D1AF7660B5CEC5D917C2F07E6AC022D2A5ED5E051D5560 "点击放大")

   以定义Component模板为例，在Abbreviation中输入模板名称，此名称作为代码模板的快捷键。在Template text中编辑模板样式，其中$符号标记用户需自行输入的部分。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/34/v3/af3H_J3oQFGrJyJb8Ynjxg/zh-cn_image_0000002229604225.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=961F491023E831FBFA87B058EEDC8F97252FC79595391DEF49176774ECC2E44C "点击放大")

   点击上图的“Change”处，选择代码模板文件。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fe/v3/UvMf4zuDRc6i-wntiF6boA/zh-cn_image_0000002194158856.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=831F85D4074CD88C33E838DE7C2F2F884D58A5C26C86D639900FC9BAF6E67C1C "点击放大")

   点击Edit variables，在弹窗中设置LABEL值的自动获取方式。

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c1/v3/C6eNyiaxRU2R4LOKTrgHow/zh-cn_image_0000002229604237.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=6DCD634DCF34DE68D59FA1B3090D87149A022CBA7908C2039C7C5C02A46EA66B)

   ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b1/v3/KyOUApTDTLKA_RtjcgooTw/zh-cn_image_0000002229758725.png?HW-CC-KV=V1&HW-CC-Date=20260612T024022Z&HW-CC-Expire=86400&HW-CC-Sign=CF9A194558B04E245D9A7BCEE0E79ED0252F906D9EA0720E665A2676BF4BA7D5)

   自定义代码模板生效。
