DevEco Studio中有一键检查未引用资源的功能，在DevEco Studio中点击菜单：Code > Analyze Code > Run Inspection by Name或使用快捷键Ctrl+Alt+Shift+I，在选项框输入Unused resources，即可进行资源检查。
