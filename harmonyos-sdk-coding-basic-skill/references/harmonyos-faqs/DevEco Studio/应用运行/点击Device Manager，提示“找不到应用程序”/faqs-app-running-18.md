**问题现象**

点击Device Manager进行登录授权时，如果提示“找不到应用程序”，则无法打开浏览器登录。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/71/v3/z9iAh5yCQg2mufkZJD7q9w/zh-cn_image_0000002229604305.png?HW-CC-KV=V1&HW-CC-Date=20260612T024413Z&HW-CC-Expire=86400&HW-CC-Sign=B1C1EAA6BA96EE6EB400AD48431A67ADD3EE8BC8330C2679A73116ACBACE45ED)

**解决措施**

该问题由默认浏览器损坏或被卸载导致，无法打开默认浏览器。

请根据[输入华为账号后，未出现“允许”按钮，浏览器界面无跳转](../../签名服务/输入华为账号后，未出现“允许”按钮，浏览器界面无跳转/faqs-signature-service-3.md)处理指导，重新设置默认浏览器后进行登录。
