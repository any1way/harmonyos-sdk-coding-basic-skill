**问题现象**

模拟器默认内存为4GB，运行过程中若内存不足，模拟器将可能出现卡顿或闪退。

**解决措施**

建议在创建模拟器时增加模拟器的运行内存（RAM）大小，请参考[创建模拟器](../../../../harmonyos-guides/编写与调试应用/使用模拟器运行应用/管理模拟器/创建模拟器/ide-emulator-create.md)。
