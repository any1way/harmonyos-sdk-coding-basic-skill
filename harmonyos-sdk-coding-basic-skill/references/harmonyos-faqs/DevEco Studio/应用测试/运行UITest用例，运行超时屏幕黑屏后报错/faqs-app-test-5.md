**问题现象**

运行UITest用例时，如果屏幕黑屏并超时，将报错提示“Tests failed”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8b/v3/yXNtH_vZSiasTInMPEsPDA/zh-cn_image_0000002229758661.png?HW-CC-KV=V1&HW-CC-Date=20260612T024534Z&HW-CC-Expire=86400&HW-CC-Sign=FBDF90ED9D5786186E7E90ACA8073911D0CCCA5C4007D021EB8DF74406CA3550)

**解决措施**

运行UITest用例时，需确保设备屏幕常亮。屏幕常亮时，用例能够成功运行。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/36/v3/rI5T7vgWRHq4AYlGhZL5-w/zh-cn_image_0000002194158788.png?HW-CC-KV=V1&HW-CC-Date=20260612T024534Z&HW-CC-Expire=86400&HW-CC-Sign=633452E7FE306E26FBDC55B03457DDA3A3F7D8AE1F17E5542946503CA09F1EA0)
