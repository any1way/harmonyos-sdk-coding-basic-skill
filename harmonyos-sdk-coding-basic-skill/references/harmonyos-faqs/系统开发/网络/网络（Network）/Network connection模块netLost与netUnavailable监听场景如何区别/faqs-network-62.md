[on('netLost')](<../../../../../harmonyos-references/网络/Network Kit（网络服务）/ArkTS API/@ohos.net.connection (网络连接管理)/js-apis-net-connection.md#onnetlost>)事件监听用于网络状态丢失。例如：Wi-Fi或移动网络断开时。

[on('netUnavailable')](<../../../../../harmonyos-references/网络/Network Kit（网络服务）/ArkTS API/@ohos.net.connection (网络连接管理)/js-apis-net-connection.md#onnetunavailable>)事件监听为网络连接成功但是发生异常无法正常访问互联网。例如：设备上显示桌面Wi-Fi图标有“!”。
